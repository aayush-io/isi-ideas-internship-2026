# IDEAS Foundation (ISI Kolkata) – Summer Internship 2026

This repository contains the data analysis notebooks, machine learning implementations, and documentation completed during my foundational internship at the Innovation in Data Engineering and Data Science (IDEAS) Foundation, Indian Statistical Institute (ISI) Kolkata.

## 📌 Project Overview
The project is divided into two core phases:
1. **Time Series Analysis of Global Temperature Anomalies (1880–2016):** Processing historical climate data to visualize long-term secular trends and variations over more than a century.
2. **Unsupervised ML Classification of Structural Digits Data:** Utilizing the `scikit-learn` digits dataset to explore high-dimensional pattern recognition and feature differentiation using unsupervised learning techniques.

## 🛠️ Core Technologies Used
* **Languages:** Python
* **Libraries:** NumPy, Pandas, Scikit-learn
* **Environment:** Jupyter Notebook

## 📈 Key Objectives Accomplished
* Cleaned and handled datasets using Pandas and NumPy pipelines.
* Conducted exploratory data analysis on historical temperature anomalies.
* Applied unsupervised machine learning workflows to evaluate feature extraction on structural image matrices.
